这张图清晰展示了Flash Attention算法的核心流程和维度信息：

**维度标注说明：**
- **B**: Batch size（批量大小）
- **L**: Sequence length（序列长度）  
- **H**: Number of heads（头数量）
- **D**: Head dimension（头维度）

**计算流程：**
1. **输入阶段**：Q, K, V张量均为[B, L, H, D]维度
2. **块状计算**：将序列分块处理，每个块保持[H, D]维度结构
3. **头维度体现**：在整个计算过程中，头维度D始终作为独立维度保留
4. **输出阶段**：最终输出保持原始维度结构

图中特别突出了头维度D在每个计算阶段的保持，以及如何通过块状处理实现内存高效的注意力计算。

    def flash_attention_v2(Q, K, V):
        # Q, K, V: [B, H, N, d]
        B, H, N, d = Q.shape
        
        # 初始化输出和统计量
        O = zeros([B, H, N, d])      # 输出 [B, H, N, d]
        m = -inf * ones([B, H, N])   # 最大值 [B, H, N]
        l = zeros([B, H, N])         # 归一化因子 [B, H, N]
        
        # 分块大小
        BLOCK_Q = 64
        BLOCK_KV = 64
        
        # 外层循环: 遍历Q块 (Flash Attention v2优化)
        for i in range(0, N, BLOCK_Q):          # i = 0, 64, 128, ..., 4032
            # 加载Q块到SRAM
            Q_i = Q[:, :, i:i+BLOCK_Q, :]       # [B, H, BLOCK_Q, d]
            
            # 内层循环: 遍历K,V块
            for j in range(0, N, BLOCK_KV):     # j = 0, 64, 128, ..., 4032
                # 加载K,V块到SRAM
                K_j = K[:, :, j:j+BLOCK_KV, :]  # [B, H, BLOCK_KV, d]
                V_j = V[:, :, j:j+BLOCK_KV, :]  # [B, H, BLOCK_KV, d]
                
                # 计算注意力分数
                S_ij = Q_i @ K_j.transpose()    # [B, H, BLOCK_Q, BLOCK_KV]
                S_ij = S_ij / sqrt(d)           # 缩放
                
                # 在线Softmax更新
                m_ij = max(S_ij, dim=-1)        # [B, H, BLOCK_Q]
                m_i_new = max(m_i, m_ij)        # 更新全局最大值
                
                # 计算指数和归一化
                P_ij = exp(S_ij - m_i_new)      # [B, H, BLOCK_Q, BLOCK_KV]
                l_ij = sum(P_ij, dim=-1)        # [B, H, BLOCK_Q]
                
                # 更新归一化因子和输出
                alpha = exp(m_i - m_i_new)      # 修正因子
                O_i = O_i * alpha + P_ij @ V_j  # [B, H, BLOCK_Q, d]
                l_i = l_i * alpha + l_ij
                
                # 更新最大值
                m_i = m_i_new
            
            # 最终归一化
            O[:, :, i:i+BLOCK_Q, :] = O_i / l_i
        
        return O  # [B, H, N, d]
